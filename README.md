# Panel TOTP de 24 Horas

Este proyecto genera y valida códigos de seguridad de 6 dígitos que cambian de forma automática cada 24 horas (a las 00:00 UTC).

## Cómo Usar

1. Abre el archivo `index.html` en cualquier navegador web.
2. **Para generar un código:**
* Escribe una palabra o frase en el campo **Clave Secreta** del panel izquierdo.
* Copia el **Código del día** generado.


3. **Para validar un código:**
* Asegúrate de que la **Clave Secreta** sea la misma en el panel derecho.
* Introduce el código de 6 dígitos en el campo correspondiente para verificar si es válido hoy.
